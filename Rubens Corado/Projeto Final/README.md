## Sobre o Projeto
Este é o projeto final da disciplina: a criação de um **Web Portfólio** do zero.
O objetivo foi aplicar os conhecimentos de HTML, CSS e JavaScript para criar uma página pessoal de apresentação.

## Tecnologias Utilizadas
* HTML5
* CSS3
* JavaScript
* VS Code

## Estrutura do Projeto
* index.html: Arquivo principal com a estrutura da página.
* /style: Pasta contendo o código de estilo (CSS).
* /script: Pasta contendo a lógica e interatividade (JavaScript).
* /img: Imagens utilizadas no layout.
